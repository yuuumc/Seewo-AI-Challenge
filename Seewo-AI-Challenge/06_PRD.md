# 06 PRD — 希沃智教π V1.0 MVP

## 1. 版本目标

> V1.0「能用且好用」—— 让数学教师在30分钟内完成原来2.5小时的批改工作，且批改质量更精细。

## 2. MVP范围定义

| 包含（IN） | 不包含（OUT） |
|-----------|---------------|
| 数学学科（高中数学首发） | 英语/语文/物理/化学 |
| 客观题全自动批改（选择/填空/判断） | 几何图形题自动批改 |
| 解答题AI辅助批改（步骤级分析） | 全自动无人值守批改 |
| 手写数学公式OCR | 化学方程式/分子式OCR |
| 错题自动归集+班级学情统计 | 年级学情仪表盘（教研端） |
| 教师端Web + 学生端微信小程序 | 教研管理端、管理后台完整版 |
| 希沃白板基础互通（单点登录） | 深度集成（课件-作业-讲评联动） |
| 基础题库（1000+高中数学题） | 变式题自动推荐 |
| 基础用户管理（手动/Excel导入） | 教务系统API自动对接 |

## 3. 核心功能规格

### F1 — 客观题自动批改
- 输入：学生手写作答照片 / 在线作答
- 处理：OCR识别 → 与答案集合匹配 → 判对错
- 输出：逐题正误、正确率统计、错误选项分布
- 验收：手写数字/字母/符号识别≥98%，单题≤3秒

### F2 — 解答题AI辅助批改
- 输入：学生手写解答题（文字+公式+图表）
- 处理：OCR→公式解析→步骤切分→逐步推理验证→错因分类→评语生成
- 输出：步骤级正误标记、错因分类标签、AI评语建议、综合评分
- 验收：AI准确率≥90%（教师修正率≤10%），单题≤30秒，低置信度(<0.7)强制提醒复核

### F3 — 手写数学公式OCR
- 验收：常见高中公式识别≥92%，复杂公式≥85%，支持中英文+公式混排

### F5 — 错题归集与学情画像
- 自动归集延迟≤1分钟，知识点标注准确率≥90%，支持时间范围筛选

### F8 — 作业全流程管理
- 支持选题/组卷/历史复用三种创建方式、班级/分层/定时三种分发方式

## 4. 非功能需求摘要

| 类别 | 关键要求 |
|------|---------|
| 性能 | 100份作业批改≤10分钟、API P95≤2秒 |
| 可用性 | SLA≥99.5%、首次学习≤15分钟、核心操作≤3步 |
| 安全 | 等保三级、TLS 1.3+AES-256、RBAC五级权限 |
| 兼容 | Chrome/Firefox/Edge、iOS15+/Android10+、希沃生态SSO |

## 5. MVP成功标准

- 10所种子学校、≥50名教师持续使用≥8周
- 教师单次批改耗时减少≥50%（自报对比）
- 周活教师率≥60%
- 教师NPS≥30

## 6. Demo 交付范围（已完成）

| 端 | 页面数 | 核心功能 |
|------|--------|---------|
