# 09 报名材料

## 项目名称
希沃智教π — AI智能作业批改系统

## 项目标语
"每个学生都被认真对待。"

## 一句话介绍
希沃智教π是专为K12教师打造的AI作业批改助手——拍照即批改、批改即分析、分析即推送。

## 项目简介（200字）
当前K12教师批改作业耗时巨大——一名高中数学教师每次批改需2.5-4小时，且反馈滞后、质量低、学情数据缺失。希沃智教π利用多Agent AI架构实现作业的智能批改——客观题全自动判定、解答题步骤级分析（定位错误步骤+错因分类+个性化评语）、AI置信度复核队列（教师只处理~15%低置信度结果）、知识点薄弱分析。学生端提供AI错题本（根因分析+知识链）、知识树（颜色可视化）、Math Coach（苏格拉底式辅导）、成长报告（趋势追踪+AI预测），形成"批改→订正→验证→闭环"的完整链路。首发数学学科，深度集成希沃硬件生态。

## 核心创新点

| 创新点 | 说明 | Demo入口 |
|--------|------|---------|
| 步骤级批改 | 解答题逐步骤分析+错因分类，非二元对错判断 | `/teacher/grade/hw_001` |
| AI置信度复核 | 标注不确定性，低置信度自动进入复核队列 | `/teacher/review/hw_001` |
| 多Agent协作 | 8个专业Agent分工，决策轨迹透明可追溯 | `/teacher/agent-trace/hw_001` |
| 订正闭环 | 批改→订正→AI验证→闭环，追踪每一步 | `/teacher/correction-loop/hw_001` |
| AI错题本 | 根因分析+知识链+分层练习，非传统错题本 | `/student/s02/error-book` |
| 知识树 | 树形可视化，颜色=掌握度，点击下钻 | `/student/s02/knowledge-tree` |
| Math Coach | 苏格拉底式引导，不给答案，追问为什么 | `/student/s02/coach` |
| 成长报告 | 趋势追踪+AI预测，不看分数看成长 | `/student/s02/growth` |
| 课堂联动 | 师生实时互动+AI即时分析+白板同步 | `/classroom` |
