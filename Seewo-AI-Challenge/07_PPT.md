# 07 演示PPT大纲

## 幻灯片结构（建议12-15页）

### 第1页 · 封面
```
           希沃智教π
    AI智能作业批改系统
  "每个学生都被认真对待"
```

### 第2页 · 痛点
```
一位高中数学教师的日常：
  带2个班 ≈ 100名学生
  每次批改 ≈ 2.5-4小时
  每周批改 ≈ 5-12小时
  58%的时间花在解答题上
  反馈延迟1-3天
  学情数据靠记忆
```

### 第3页 · 产品定位
```
希沃智教π = AI作业批改助手
  拍照即批改 → 批改即分析 → 分析即推送

  一线教师：批改时间减半，每道题获得步骤级诊断
  教研组长：年级学情实时可见，数据驱动教研
  学生：即时反馈，知道"错在哪、为什么、怎么改"
```

### 第4页 · 核心功能
```
        AI批改引擎
  客观题全自动  +  解答题步骤级分析

  ┌─ 选择/填空/判断        ─┐
  │  OCR识别·答案匹配·自动判分 │
  └──────────────────────────┘
  ┌─ 解答题  ──────────────────┐
  │  步骤切分→逐步验证→错因分类  │
  │  →个性化评语→AI置信度标注    │
  └────────────────────────────┘
```

### 第5页 · 步骤级分析（核心差异化）
```
  传统批改：        AI批改：
   Q5  ✗ 4/15       Q5  步骤① ✓ 求导正确
                      ② ✗ 配方法错误 → 漏写+a²项
                      ③ ✗ 因②错误无法继续
                      ④ ✗ 结论缺失
                    错因：🔴 计算错误
                    建议：复习完全平方公式
```

### 第6页 · 产品截图（Demo展示）
```
  [教师端Dashboard]    [AI批改结果·步骤级分析]
  [AI复核队列]         [多Agent决策追踪]
  [学生首页]           [AI错题本·知识链]
  [知识树]             [Math Coach对话]
  [成长报告·AI预测]    [课堂互动]
```

### 第7页 · 教研管理价值
```
  教研组长视角：
    知识点热力图 → 精准定位薄弱点
    班级横向对比 → 发现教学差异
    教师行为数据 → 量化批改质量
    一键生成报告 → 告别Excel手工统计
```

### 第8页 · 与希沃生态的整合
```
  备课(希沃白板) → 授课(智慧黑板) → 作业(智教π) → 学情(数据看板)
                └─── 教学评一体化 ────┘
```

### 第9页 · 竞争格局

## 附录：Demo演示要点

1. **开场**：展示教师批改场景的痛点数据（2.5小时的作业批改）
2. **核心流**：教师拍照上传→AI自动批改→步骤级分析→发布结果
3. **亮点**：聚焦解答题的步骤级诊断（不是简单的对/错）
4. **学生侧**：展示学生如何看到反馈、如何订正
5. **学情**：教研组长看到的班级学情仪表盘
6. **结尾**：回到"每个学生都被认真对待"
